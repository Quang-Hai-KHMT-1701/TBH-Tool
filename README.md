# TaskbarHero Drop Tool

TaskbarHero Drop Tool giúp xem trước danh sách vật phẩm trong rương và theo dõi các vật phẩm quan trọng khi chơi TaskbarHero.

## Cách sử dụng

1. Mở `TaskbarHero.exe`.
2. Mở `TaskbarHero Drop Tool.exe`.
3. Nhập key bản quyền nếu ứng dụng yêu cầu kích hoạt.
4. Bấm `Bật` để kết nối với game.
5. Chuyển map hoặc mở rương trong game để danh sách rương được cập nhật.

Sau lần kích hoạt đầu tiên, ứng dụng sẽ tự ghi nhớ key trên máy hiện tại. Những lần mở sau bạn không cần nhập lại key nếu key vẫn còn hợp lệ.

- Hãy mở game trước khi bấm `Bật`.
- Nếu ứng dụng không kết nối được, hãy tắt rồi mở lại cả game và tool.
- Không đổi tên file exe nếu bạn không chắc chắn.
- Không chia sẻ key bản quyền cho thiết bị khác.

## Hỗ trợ

Khi cần hỗ trợ, hãy gửi ảnh chụp màn hình ứng dụng kèm mô tả ngắn tình huống bạn gặp phải.
